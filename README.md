Gameplay
The game is a classic Rock Paper Scissors game between you and the computer. Choose between Rock, Paper, or Scissors, and the computer will randomly select its choice. The winner is determined by the following rules:

Rock beats Scissors
Scissors beats Paper
Paper beats Rock
A tie occurs if both players select the same option
Features
Single player game against the computer
Random computer choice
Clear winner determination
Hand image display for your choice and computer's choice
Technologies Used
HTML: For structuring the content of the web page.
CSS: For styling the game elements and outcome display.
JavaScript: For implementing the game logic, computer selection, and winner determination.
